# MusicPlayer

Một dự án trình phát nhạc đơn giản dùng để học và demo (OJT-2).

## Mô tả

Dự án này là một giao diện web cho ứng dụng nghe nhạc, phát triển bằng React + TypeScript. Nó bao gồm các trang hiển thị album, nghệ sĩ, thể loại và thanh điều khiển phát nhạc.

## Tính năng chính

- Hiển thị album, nghệ sĩ và thể loại
- Danh sách album nổi bật / mới phát hành / xu hướng
- Thanh điều khiển phát nhạc (play/pause, next, prev)
- Modal đăng nhập/đăng ký (giao diện)

## Công nghệ

- React
- TypeScript
- Vite
- Tailwind CSS

## Chạy dự án cục bộ

1. Cài đặt phụ thuộc

```bash
npm install
```

1. Chạy ở chế độ phát triển

```bash
npm run dev
```

1. Tạo build để deploy

```bash
npm run build
```

## Cấu trúc cơ bản

- `src/` : mã nguồn chính (components, pages, store, api...)
- `public/` / `index.html` : entry
